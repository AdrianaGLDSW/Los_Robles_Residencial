# Los Robles Residencial

Proyecto: Aplicación web/mobile enfocada en la gestión administrativa del Residencial Los Robles, desarrollada bajo metodología Scrum.


Objetivo del proyecto:

Desarrollar una plataforma que permita optimizar la administración del residencial mediante módulos funcionales para residentes y administradores.


Funcionalidades principales:
Residentes
- Inicio de sesión
- Recuperación de contraseña
- Historial de pagos
- Descarga de comprobantes
- Notificaciones
- Tablero de avisos
- Botón de emergencias

Administración
- Gestión de residentes
- Dashboard financiero
- Publicación de avisos
- Envío de notificaciones masivas
- Gestión de reportes


Tecnologías utilizadas:
- HTML5
- CSS3
- JavaScript
- Firebase
- GitHub Pages

Metodología de trabajo:

El proyecto fue desarrollado utilizando metodología ágil Scrum mediante:
- Product backlog
- Sprint planning
- Sprint review
- Sprint retrospective
- Daily scrum meetings

Mejoras implementadas:

Sprint de mejoras. Durante las reuniones de seguimiento se implementaron las siguientes mejoras:
- Optimización del diseño responsivo.
- Corrección de validaciones en formularios.
- Mejora visual de interfaces.
- Reestructuración del Product Backlog.
- Incorporación de métricas de avance.
- Corrección de bugs detectados por QA.
- Actualización de documentación técnica.
